## ГЛАВА 1. НАЧАЛО ##
-- Offensive security --
Это способ защиты при котором думаешь как злоумышленник, пытаясь продумать какими способами можно проникнуть, навредить, 
украсть важные данные системы.
Утилита dirb - примитивный брут форс метод по поиску скрытых эндпоинтов на сокете.
-- Defensive security --
Security Operations Center (SOC) - команда занимающаяся мониторингом систем, поиском подозрительных действий и устранением угроз.
Malware - любая вредоносная программа.
-- Search engines --
Поиск информации довльно важная часть современной жизни.
Полезны поисковые движки:

Поиск любых IoT устройств - https://www.shodan.io
Internet assets - Censys
Проверка файлов, ссылок на вредоностность - https://www.virustotal.com
Проверка email на утечку данных - https://haveibeenpwned.com
Узнать свой IP в интернете - https://www.myip.com
